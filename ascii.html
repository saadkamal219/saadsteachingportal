What is ASCII ?

ASCII stands for American Standard Code for Information Interchange. ASCII is a 8 bit binary code. Each and every character in the keyboard has unique ASCII code. Sometimes we need to send more bits at a time, 9 bits. For that case the communicators use EBCDIC code. It stands for Extended Binary Coded Decimanl Interchange Code.
